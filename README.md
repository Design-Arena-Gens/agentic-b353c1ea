## Creator Command Hub

An operator dashboard for running a YouTube channel like a product launch. Plan your publishing rhythm, track channel vitals, and spin up fresh video concepts with a built-in idea lab.

### Features

- **Channel cockpit:** High-level metrics, publishing rituals, and operating cadence for predictable uploads.
- **Content pillars:** Visualize the promise, cadence, and focus points that guide every storyline.
- **Idea generator:** Produce a hook, outline, monetization angle, and supporting Shorts concept with one click.
- **Production pipeline:** Keep upcoming releases, trends to riff on, and reflection rituals in one view.

### Tech Stack

- [Next.js](https://nextjs.org/) App Router
- [React](https://react.dev/) 19
- [Tailwind CSS](https://tailwindcss.com/) v4

### Local Development

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to explore the hub.

### Production Build

```bash
npm run build
npm run start
```

This project is prepped for seamless deployment on Vercel.
